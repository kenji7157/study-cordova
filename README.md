コルドバチュートリアルの学習用リポジトリ
http://ccoenraets.github.io/cordova-tutorial/create-cordova-project.html


iOSの動作確認手順
```
$ cordova platform add ios@latest
$ cordova build ios
$ cordova emulate ios
```

androidの動作確認手順
```
$ cordova platforms add android
(実機の場合)
$ cordova run android
(エミュレータの場合)
$ cordova emulate android
```
